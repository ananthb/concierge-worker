Concierge

Multi-channel messaging automation for small businesses. WhatsApp and Instagram auto-replies, email routing with smart rules, a unified Discord inbox, embeddable lead capture forms, and credit-based billing. All in one self-hosted Cloudflare Worker.

Hosted Service

Don't want to self-host? concierge.calculon.tech runs this exact stack as a managed service. Sign up, connect your channels, and start auto-replying in minutes. 100 free replies every month, paid credit packs and email subdomains when you need more.

Features

  • WhatsApp Auto-Reply: reply instantly to incoming messages with static text or AI-generated responses via the Meta Business API
  • Instagram DM Auto-Reply: auto-respond to Instagram Direct Messages via Facebook Pages API
  • Email Routing: catch-all email with flexible routing rules using glob patterns. Actions include drop, spam reject, forward email (with reverse-alias reply routing), forward to Discord, and AI-generated replies with approval workflow
  • Discord Relay: unified inbox in Discord where messages from any channel are forwarded with Reply, Approve, and Drop buttons. Replies in Discord route back through the originating channel
  • Discord Slash Commands: /status, /domains, and /rules for managing your setup from Discord
  • Lead Capture Forms: embeddable phone number forms that trigger WhatsApp messages on submission
  • Unified Message Pipeline: all channels produce a common InboundMessage type, logged to a unified messages table (metadata only, no message content stored)
  • Onboarding Wizard: 6-step guided setup for new tenants
  • Management Panel: platform operator dashboard at /manage/* behind Cloudflare Access for tenant CRUD, billing controls, and audit logs
  • Billing: prepaid reply credits with 100 free per month. Purchased credits never expire. Razorpay integration for paid credit packs
  • Multi-Tenant Auth: Google and Facebook OAuth with multi-provider account linking
  • Privacy-First: no message content stored. GDPR data deletion endpoint. AGPL-3.0 licensed

Built With

  • Cloudflare Workers (Rust + WebAssembly)
  • Cloudflare D1 (SQLite), KV, Workers AI, and Email Routing
  • Meta WhatsApp Business API + Facebook Login + Instagram Graph API
  • Discord Interactions API (slash commands, buttons, modals)
  • Razorpay payment gateway

Getting Started (Self-Hosted)

Click the deploy button above to deploy to Cloudflare Workers. The deploy flow will fork the repo and provision the Worker, D1 database, and KV namespace automatically. Then follow the Deployment Guide (or the Setup guide) to configure secrets and webhooks.
