Concierge

Messaging automation for small businesses. WhatsApp auto-replies, Instagram DM auto-replies, and embeddable lead capture forms — zero effort.

Features

  • WhatsApp Auto-Reply — Reply instantly to incoming messages with static text or AI-generated responses
  • Instagram DM Auto-Reply — Auto-respond to Instagram Direct Messages via your business account
  • Lead Capture Forms — Embeddable phone number forms that trigger WhatsApp messages on submission

Built With

  • Cloudflare Workers (Rust + WebAssembly)
  • Cloudflare D1, KV, and Workers AI
  • Meta WhatsApp Business API
  • Facebook Login + Instagram Graph API
